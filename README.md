# App học từ vựng N2/N3 - Active Recall, có ví dụ và phiên âm cả câu

Ứng dụng web tĩnh, chạy offline. Phiên bản này bổ sung ví dụ cho toàn bộ dữ liệu N2/N3 và hiển thị:

- Ví dụ tiếng Nhật
- Phiên âm cả câu bằng hiragana
- Furigana dạng `漢字（かな）`
- Nghĩa tiếng Việt

## Dữ liệu
- Tổng: 4076 mục
- Có ví dụ từ tài liệu nguồn: 2444 mục
- Ví dụ bổ sung tự tạo cho mục chưa có ví dụ nguồn: 1632 mục

## Cách dùng GitHub Pages
Upload toàn bộ thư mục này lên repository GitHub, đảm bảo file chính là `index.html`, sau đó bật Settings → Pages → Deploy from branch → main → /root.
